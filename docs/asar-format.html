Asar format

MAGIC:
ASAR\r\n
PAYLOAD:
file1 content
file2 content
file3 content
file4 content
file5 content
HEADER:
{ "version": 1, "files": { "file1.txt": {"offset": 6, "size": 13}, "dir1": { "files": { "file2.txt": {"offset": 19, "size": 13}, "dir3": { "files": { "file3.txt": {"offset": 32, "size": 13} }} }}, "dir2": { "files": { "file4.txt": {"offset": 45, "size": 13}, "file5.txt": {"offset": 58, "size": 13} }} } }
HEADER-SIZE:
UINT32 (4 byte)
CHECKSUM: (containes everything up to this point)
MD5 (16 byte)
ARCHIVE-SIZE:
UINT64 (8 byte)
