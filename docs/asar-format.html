Asar format

MAGIC:
ASAR\r\n
PAYLOAD:
file1 content
file2 content
file3 content
file4 content
file5 content
HEADER:
{ "version": 1, "files": { "file1.txt": {"size": 13, "offset": 6}, "dir1": { "files": { "file2.txt": {"size": 13, "offset": 19}, "file3.txt": {"size": 13, "offset": 32} } }, "dir2": { "file4.txt": {"size": 13, "offset": 45}, "file5.txt": {"size": 13, "offset": 58} } } }
HEADER-SIZE:
UINT64 (8 byte)
CHECKSUM:
MD5 (16 byte)
ARCHIVE-SIZE:
UINT64 (8 byte)
